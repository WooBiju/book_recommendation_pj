# 📚맞춤형 도서 추천 시스템

## 프로젝트 개요
- 사용자 선호 기반 도서 추천 시스템 구현
- 목표 : 단순 CRUD 를 넘어서 개인화 추천 시스템 백엔드 개발 경험을 목표로 구현

## 기술 스택
| 분류 | 사용 기술 |
|------|-----------|
| Language | Java 17, Python 3.10 |
| Framework | Spring Boot, FastAPI |
| DB | MySQL, Redis |
| Build Tool | Gradle |
| Infra | EC2 (예정), Docker (예정) |
| ETC | Swagger, Postman |
